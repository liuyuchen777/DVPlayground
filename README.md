# Data Visualization Playground

## dv-main

Use `d3.js` and `React.js` to build data visualizations.

[Data Visualization with D3, JavaScript, React - Full Course](https://www.youtube.com/watch?v=2LhoCfjm8R4&t=12467s)

```
cd dv-main
npm i
npm start
```
